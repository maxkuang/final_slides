Monge's Problem¶

  • The optimal transport problem, as proposed originally by Monge in 1781, addresses the displacement of a pile of soil between two locations with minimal cost.
  • Given the cost $c(\mathbf{x},\mathbf{y})$ of moving a unit mass from point $\mathbf{x}$ to point $\mathbf{y}$, one seeks the map $\mathbf{y} = \mathbf{f}(\mathbf{x})$ that minimizes total cost. Suppose the two piles of soil are two probability measures $\mu_1$ and $\mu_2$, $$\inf_{\mathbf{f}_\sharp\mu_1 = \mu_2} \int_{\mathbb{R}^d} c(\mathbf{x}, \mathbf{f}(\mathbf{x})) d\mu(\mathbf{x}),$$ where $\mathbf{f}_\sharp\mu_1$ represents the pushforward measure.

Kantorovich's Formulation¶

  • Kantorovich relaxed Monge's problem, allowing the movement of soil from one to multiple locations and vice versa.
  • Denoting the mass transported from $\mathbf{x}$ to $\mathbf{y}$ by $\pi(\mathbf{x},\mathbf{y})$, the minimization problem can be rewritten as $$\inf_{\pi\in\Pi_{\mu_1,\mu_2}} \int c(\mathbf{x},\mathbf{y})\pi(\mathbf{x},\mathbf{y})d\mathbf{x} d\mathbf{y},$$ where $\Pi_{\mu_1,\mu_2}$ is the set of all the transfer plans $\pi(\mathbf{x},\mathbf{y})$ satisfying the marginal constraints $$\int\pi(\mathbf{x},\mathbf{y})d\mathbf{y} = \mu_1(\mathbf{x}),$$ $$\int\pi(\mathbf{x},\mathbf{y})d\mathbf{x} = \mu_2(\mathbf{y}).$$

Kantorovich's Dual Theorem¶

$$ \inf_{\pi\in\Pi_{\mu_1,\mu_2}} \int c(\mathbf{x},\mathbf{y})\pi(\mathbf{x},\mathbf{y})d\mathbf{x} d\mathbf{y}=\sup_{\phi(\mathbf{x}) + \psi(\mathbf{y}) \leq c(\mathbf{x},\mathbf{y})} \int\phi(\mathbf{x})d\mu_1(\mathbf{x}) + \int\psi(\mathbf{y})d\mu_2(\mathbf{y}).$$
  • With minor conditions, one can prove the solution of Kantorovich's formulation is unique and equals to the solution of Monge's problem.
  • For instance $c(\mathbf{x},\mathbf{y}) = \|\mathbf{x} - \mathbf{y}\|^2$ in $\mathbb{R}^d$, $\mu$ absolutely continuous with respect to Lebesgue. (Gangbo and McCann 1996)

Monge-Ampere PDE¶

  • The optimal map from $\mu$ to $\nu$ is in the form of gradients: $$\mathbf{y} = \nabla\phi(\mathbf{x})$$ where $\phi$ is the function in the dual problem, and must be convex in $\mathbf{x}$.
  • The optimal $\phi$ satisfies the Monge-Ampere PDE:
$$ \det{(\nabla^2\phi(\mathbf{x}))} d\mu_2(\nabla\phi(\mathbf{x})) = d\mu_1(\mathbf{x}).$$
  • In an integral form
$$ \int_{\mathbb{R}^d} h(\nabla\phi(\mathbf{x})) d\mu_1(\mathbf{x}) = \int_{\mathbb{R}^d} h(\mathbf{y})d\mu_2(\mathbf{y})$$

Wasserstein Distance and Wasserstein Barycenter¶

  • For $c(\mathbf{x},\mathbf{y}) = \|\mathbf{x} - \mathbf{y}\|^2$ in $\mathbb{R}^d$, the transportation cost induces the 2-Wasserstein distance: $$ W_2(\mu_1, \mu_2) = \left(\min_{\pi\in\Pi_{\mu_1,\mu_2}}\int_{\mathbb{R}^d} \|\mathbf{x}-\mathbf{y}\|^2d\pi(\mathbf{x},\mathbf{y})\right)^{\frac{1}{2}},$$
  • It is a metric of the space of measures (Villani 2008):
$$ P_2(\mathbb{R}^d) := \left\lbrace \mu\in P(\mathbb{R}^d); \quad \int_{\mathbb{R}^d} \|\mathbf{x}\|^2 d\mu(\mathbf{x})<+\infty\right\rbrace .$$

Wasserstein Barycenter Measure¶

  • Center of measures. $\mu_1, \mu_2, \cdots, \mu_K$ and weights $w_1, w_2,\cdots, w_K$.
$$ \bar{\nu} = \underset{\nu\in P_2(\mathbb{R}^d)}{\mathrm{argmin}} \sum_{k=1}^K w_k W^2_2(\mu_k, \nu).$$
  • A natural extention to barycenter of points:
$$ \bar{\mathbf{x}} = \sum_{k=1}^K w_k\mathbf{x}_k = \underset{\mathbf{x}}{\mathrm{argmin}} \sum_{k=1}^K w_k \|\mathbf{x} - \mathbf{x}_k\|^2 $$
  • More natural than the simple average

A Simpler Question¶

  • Assuming that we already have a black box solver for the optimal transport problem, find an efficient algorithm that uses it to solve the barycenter problem.
  • Wasserstein barycenter problem:
$$\bar{\nu} = \underset{\nu\in P_2(\mathbb{R}^d)}{\mathrm{argmin}} \sum_{k=1}^K w_k W^2_2(\mu_k, \nu)$$
  • Start from some initial measure $\nu = \nu_0$;
  • Approach $\bar{\nu}$ through an iterative procedure:
$$\nu_{n} \xrightarrow{\hspace{0em}?} \nu_{n+1}$$

  • Suppose $\mathbf{s}_k(\mathbf{y})$ is the optimal map from $\nu$ to $\mu_k$. The total cost can be written as
\begin{equation} \sum_{k=1}^K w_k\int \|\mathbf{y} -\mathbf{s}_k(\mathbf{y})\|^2 d\nu(\mathbf{y}) = \int\left[\sum_{k=1}^K w_k\|\mathbf{y} - \mathbf{s}_k(\mathbf{y})\|^2\right]d\nu(\mathbf{y}). \end{equation}
  • We move $\mathbf{y}\rightarrow \mathbf{f}(\mathbf{y})$, such that
\begin{equation} \sum_{k=1}^K w_k\|\mathbf{y} - \mathbf{s}_k(\mathbf{y})\|^2 \geq \sum_{k=1}^K w_k\|\mathbf{f}(\mathbf{y}) - \mathbf{s}_k(\mathbf{y})\|^2 \end{equation}
  • Candidate for $\mathbf{f}(\mathbf{y})$:
\begin{equation} \mathbf{f}(\mathbf{y}) = \sum_{k=1}^K w_k\mathbf{s}_k(\mathbf{y}) \end{equation}

A Simpler Question:
Basic Barycenter Algorithm

  1. Set $\nu = \nu_0$ where $\nu_0\in P_2(\mathbb{R}^d)$ is an arbitrary initial measure;
  2. Find the optimal maps $\mathbf{s}_k$ from $\nu$ to $\mu_k$ ($k = 1, 2,\cdots, K$);
  3. Define $\mathbf{f}(\mathbf{y}) = \sum_{k=1}^K w_k\mathbf{s}_k(\mathbf{y})$;
  4. Set $\nu = \mathbf{f}_\sharp \nu$ and return to step 2.

A Simpler Question:
Basic Barycenter Algorithm

  • The algorithm reduces the total transportation cost at every step;
  • The algorithm stops only when:
\begin{equation} \mathbf{y} = \sum_{k=1}^K w_k\mathbf{s}_k(\mathbf{y}); \end{equation}
  • This means, barycenter measure $\bar{\nu}$ must satisfy the above equation.

Algorithms: McCann Interpolant and Barycenter¶

  • A special 2-barycenter problem
\begin{equation} \min_{\nu\in P_2(\mathbb{R}^d)} \frac{1}{2}W_2^2(\mu_1, \nu) + \frac{1}{2}W_2^2(\mu_2, \nu), \end{equation}
  • Its solution coincides with the McCann interpolant measure.
$$\mathbf{y} = \frac{1}{2}\mathbf{s}_1(\mathbf{y}) + \frac{1}{2}\mathbf{s}_2(\mathbf{y}) $$
  • The optimal map from $\mu_1$ to $\mu_2$ is given by
$$ \mathbf{z} = \mathbf{s}_2(\mathbf{s}^{-1}_1(\mathbf{x}))$$
  • With more segments on the McCann Interpolants
\begin{equation} \min_{\substack{\nu_0,\nu_1,\cdots,\nu_K\in P_2(\mathbb{R}^d)\\ \nu_0 = \mu_1,\nu_K = \mu_2}} \sum_{k=1}^K w_kW_2^2(\nu_k,\nu_{k-1}) . \end{equation}
  • Define $\mathbf{s}_k$ as the optimal from $\nu_{k-1}$ to $\nu_k$, and
$$\lambda_k = \sum_{i=1}^k \frac{1}{w_i}\left/ \sum_{i=1}^K \frac{1}{w_i}\right.$$$$ \mathbf{s}^c = \mathbf{s}_K\circ\mathbf{s}_{K-1}\circ\cdots\circ\mathbf{s}_1.$$
  • $\mathbf{s}^c$ should be the optimal map from $\mu_1$ to $\mu_2$.

Algorithms: Theoretical Optimal Transport Algorithm (TOT)¶

  1. Let $\nu_0 = \mu_1$ and $\nu_K = \mu_2$. Set $\nu_k$ ($k=1,2,\cdots, K-1$) to arbitrary initial measures in $P_2(\mathbb{R}^d)$;
  2. Find the optimal maps $\mathbf{s}_k$ from $\nu_{k-1}$ to $\nu_k$ ($k = 1, 2,\cdots, K$);
  3. Define $\mathbf{s}^c$ and the weights $\lambda_k$;
  4. For $k=1,2,\cdots, K-1$, update $\nu_k$ to $$ \nu_k = [\lambda_k\mathbf{s}^c + (1 - \lambda_k) \mathbf{id}]_{\sharp} \mu_1 $$ and return to step 2.

Algorithms: Illustration¶

Algorithms: Composite Theoretical Barycenter Algorithm (CTB)¶

  1. Set $\nu = \nu_0$ where $\nu_0\in P_2(\mathbb{R}^d)$ is an arbitrary initial measure;
  2. Run steps 1,2,3 of the TOT algorithm once to find a map $\mathbf{s}_k^c$ from $\nu$ to $\mu_k$ for each $k=1,2,\cdots,K$;
  3. Define the map $\mathbf{f}^c$ as: $$\mathbf{f}^c(\mathbf{y}) = \sum_{k=1}^K w_k \mathbf{s}_k^c(\mathbf{y});$$
  4. Update $\nu$ and $\mathbf{s}_k^c$: $$\nu = \mathbf{f}^c_\sharp\nu;$$ $$\mathbf{s}_k^c = \mathbf{s}_k^c\circ(\mathbf{f}^c)^{-1};$$
  5. Run step 4, 2, 3 of TOT once for each pair $(\nu, \mu_k)$ to update $\mathbf{s}_k^c$ and return to step 3.

Convergence Theorems:
A Property of the Barycenter Measure

  • Theorem 1(Agueh and Carlier 2011). Consider the barycenter problem with measures $\mu_1,\mu_2,\cdots,\mu_K \in P_2(\mathbb{R}^d)$ and positive normalized weights $w_1, w_2,\cdots, w_K$. If one of the measure is absolute continuous, a measure $\nu$ is the barycenter if and only if for almost all $\mathbf{y}\in\mathrm{supp}(\nu)$, $$\mathbf{y}= \sum_{k=1}^K w_k\mathbf{s}_k(\mathbf{y}),$$ where $\mathbf{s}_k$ is the optimal map from $\nu$ to $\mu_k$.

Sketch of Proof¶

  • If $\nu$ is the barycenter (Necessity):
$$ \begin{array}{rl} \displaystyle \sum_{k=1}^K w_k W^2_2(\nu,\mu_k)&\geq \displaystyle\int\left[\sum_{k=1}^K w_k\|\mathbf{f}(\mathbf{y}) - \mathbf{s}_k(\mathbf{y})\|^2\right]d\nu(\mathbf{y})\\ &\displaystyle=\int\left[\sum_{k=1}^K w_k\|\tilde{\mathbf{y}} - \mathbf{s}_k(\mathbf{f}^{-1}(\tilde{\mathbf{y}}))\|^2\right]d\mathbf{f}_\sharp\nu(\tilde{\mathbf{y}})\\ &\displaystyle\geq \sum_{k=1}^K w_k W^2_2(\mathbf{f}_\sharp\nu,\mu_k). \end{array} $$
  • You must have $\mathbf{y} = \mathbf{f}(\mathbf{y})$ almost everywhere.

Sketch of Proof¶

  • When $\mathbf{y}= \sum_{k=1}^K w_k\mathbf{s}_k(\mathbf{y})$: (Sufficiency) $$\mathbf{y} = \sum_{k=1}^K w_k \mathbf{}s_k(\mathbf{\mathbf{y}}) = \sum_{k=1}^K w_k \nabla \left[\frac{\|\mathbf{y}\|^2}{2} - \frac{\psi^*_k(\mathbf{y})}{w_k}\right] = \mathbf{y} - \nabla\sum_{k=1}^K \psi^*_k(\mathbf{y}).$$ This shows $\sum_{k=1}^K \psi^*_k(\mathbf{y})$ is a constant, from the dual formulation of the multidimensional optimal transport problem, we know that such $\psi_k$ achieves the maximum (minimum), thus $\nu$ has to be the barycenter.

Convergence Theorems:
A Property of the McCann Interpolant

  • Theorem 2. Consider measures $\nu_0,\nu_1,\cdots,\nu_K$ and positive normalized weights $w_1, w_2,\cdots, w_K$. Suppose $\mathbf{s}_k$ is the optimal map from $\nu_{k-1}$ to $\nu_{k}$. Measures $(\nu_0,\nu_1,\cdots,\nu_K)$ minimize $$ \begin{equation} \min_{\substack{\nu_0,\nu_1,\cdots,\nu_K\in P_2(\mathbb{R}^d)\\ \nu_0 = \mu_1,\nu_K = \mu_2}} \sum_{k=1}^K w_kW_2^2(\nu_k,\nu_{k-1}) . \end{equation} $$ if and only if for almost all $\mathbf{y}\in\mathrm{supp}(\mu_1)$ and all $k = 1, 2,\cdots,K$, $$ \begin{equation} \mathbf{s}_k\circ\cdots\circ\mathbf{s}_2\circ\mathbf{s}_1(\mathbf{}y) = (1 - \lambda_k)\mathbf{y} + \lambda_k\mathbf{s}^c(\mathbf{y}). \end{equation} $$

Convergence Theorems: $L$-descending Map¶

  • Definition ($L$-descending Map) .Suppose $\mathcal{F}$ is a map between measures: $$ \begin{equation} \mathcal{F}:P_2(\mathbb{R}^d)\rightarrow P_2(\mathbb{R}^d) \end{equation} $$ and $L:P_2(\mathbb{R}^d)\rightarrow \mathbb{R}$ is a cost function with a lower bound. We call $\mathbf{f}$ a $L$-descending map if it satisfies the following conditions:
    1. $\mathcal{F}$ is a continuous map with respect to the $W_2$ metric on $P_2(\mathbb{R}^d)$.
    2. For arbitrary $\pi\in P_2(\mathbb{R}^d)$, we have $$ \begin{equation} L(\mathcal{F}\pi) \leq L(\pi) \end{equation} $$ and equality holds if and only if $$ \begin{equation} \mathcal{F}\pi = \pi = \pi^* \end{equation} $$ where $\pi^*$ is a minimizer of $L$.

Convergence Theorems: $L$-descending Map¶

  • Theorem 3 .Let $\mathcal{F}$ be an $L$-descending map. We define a sequence $\{\pi^n\}_{n = 1, 2,...}$ through $$\pi^{n+1} = \mathcal{F}\pi^n,\quad n = 0,1,...$$ where $\pi^0 \in P_2(\mathbb{R}^d)$ is an arbitrary initial measure. If the following two conditions are satisfied:
    1. $\{\pi^n\}_{n = 1, 2,...}$ is sequentially compact with respect to the $W_2$ metric,
    2. $L$ is continuous with respect to the $W_2$ metric and it has a unique minimizer $\pi^*$, then $$\pi^n\xrightarrow[]{W_2} \pi^*$$

Convergence Theorems:
General Descending Optimal Transport Algorithm

  • Definition (GDOT). Consider an iterative algorithm that updates a transfer plan $\pi\in\Pi_{\mu_1,\mu_2}$: $$ \begin{equation} \pi^{n+1} = \mathcal{F}\pi^n,\quad n = 0,1,\cdots \end{equation} $$ We call an algorithm a general descending optimal transport algorithm if the associated iteration map $\mathcal{F}$ is a $L$-descending map in which the cost function $L(\pi)$ is the total transport cost for optimal transport.
  • Theorem 4. Suppose the two target measures $\mu_1$ and $\mu_2$ are in $P_2(\mathbb{R}^d)$ and the optimal transfer plan is unique. Then the GDOT algorithm converges to the optimal transfer plan $\pi^*$ in the $W_2$ metric.

General Descending Barycenter Algorithm

  • Definition (GDB). Given an arbitrary $L$-descending map $\mathcal{F}$ for the GDOT algorithms, we define the map: $$ \begin{equation} \mathcal{G}:\Pi_{\mu_1,\cdots,\mu_2,\cdots,\mu_K}\rightarrow \Pi_{\mu_1,\cdots,\mu_2,\cdots,\mu_K} \end{equation} $$ and use it to update the current multimarginal measure $\hat{\pi}^n$: $$ \begin{equation} \hat{\pi}^{n+1} = \mathcal{G}\hat{\pi}^{n}. \end{equation} $$ We call such an algorithm a general descending barycenter algorithm if $\mathcal{G}$ is induced by $\mathcal{F}$ through the following steps:

Convergence Theorems:
General Descending Barycenter Algorithm

  1. Derive $\nu^n$ from $\hat{\pi}^n$: $$ \nu(E) := \hat{\pi}\left( \{(\mathbf{x}_1,\cdots,\mathbf{x}_K)| \sum_{k=1}^K w_k\mathbf{x}_k \in E\}\right),\text{ for all Borel sets } E. $$
  2. For $k = 1,2,\cdots, K$, set the current transfer plan $\pi_k^n$ to $$ \begin{equation} \pi_k^n(E_\nu\times E_{k}) := \nu(E_\nu)\cdot\hat{\pi}^n\left( \{(\mathbf{x}_1,\cdots,\mathbf{x}_K)| \mathbf{x}_k \in E_{k}\}\right), \text{ for all Borel sets } E_\nu, E_{k}; \end{equation} $$
  3. Apply the map $\mathcal{F}$ to all $\pi_k^n$, $$ \begin{equation} \pi_k^{n+1} = \mathcal{F}\pi_k^n \end{equation} $$ and define the conditional measure $\pi_k^{n+1}|\mathbf{y}$;
  4. Set $\hat{\pi}^{n+1}$ as the product measure: $$ \begin{equation} \hat{\pi}^{n+1}(E_1\times E_2\times\cdots\times E_K) = \int_{\mathbb{R}^d} \left[\prod_{k=1}^K\pi_k^{n+1}| \mathbf{y}(E_k)\right]d\nu(\mathbf{y}). \end{equation} $$
  • Theorem 5. The general descending barycenter algorithm has the following properties:
    1. $\mathcal{G}$ is a $L$-descending map, where the cost function $L$ is the barycenter cost in the multimarginal formulation: $$ \begin{equation} L(\hat{\pi}) = \int_{\mathbb{R}^{dK}}\left(\sum_{k=1}^Kw_k\|\mathbf{x}_k - \bar{\mathbf{x}}\|^2\right)d\hat{\pi}(\mathbf{x}_1,\cdots,\mathbf{x}_K), \quad \hat{\pi}\in\Pi_{\mu_1,\mu_2,\cdots,\mu_K}; \end{equation} $$
    2. Suppose the barycenter problem has a unique solution, the GDB algorithm always converges to the minimizer $\hat{\pi}^*$ in $W_2$; the corresponding $\nu^*$ is the barycenter.

Sample-based Perspective: Goals¶

Sample-based Optimal Transport¶

  • Only sample sets are available, data distributions are unknown: i.i.d. $\{\mathbf{x}_i\}_{i=1}^{N_x}$ and $\{\mathbf{y}_i\}_{i=1}^{N_y}$.
  • We would like to define and find the optimal map from $\{\mathbf{x}_i\}_{i=1}^{N_x}$ to $\{\mathbf{y}_i\}_{i=1}^{N_y}$.
  • An existing framework:

Sample-based Optimal Transport¶

  • Discrete form 1: $\mu_1$ and $\mu_2$ are modeled simply using dirac measures of data points.
  • Discrete form 2: $\mu_1$ and $\mu_2$ are derived using density estimation
  • because if we think the unknown distribution is a continuous measure, the solution should be a map.
  • In many data applications, a map gives a more definite answer than a transfer plan.

Sample-based Optimal Transport:
Two Elements of Optimal Map

  • We define the optimal map directly from data!
  • A map $\mathbf{f}$ pushes measure $\mu_1$ to $\mu_2$:
$$\begin{equation}\mathbf{f}_\sharp\mu_1 = \mu_2\end{equation}$$
  • $\mathbf{f}$ minimizes the transport cost, is a gradiant of a convex function $\phi$:
$$ \begin{equation}\min_{\mathbf{f}} \int \|\mathbf{x} - \mathbf{f}(\mathbf{x})\|^2 d\mu_1(\mathbf{x}),\quad \mathbf{f}(\mathbf{x}) = \nabla\phi(\mathbf{x})\end{equation}$$

Sample-based Optimal Transport:
Sample-based Equivalence

  • When we only have samples, we need to define:
$$ \{\nabla\phi(\mathbf{x}_i)\}_{i=1}^{N_x}=\{\mathbf{y}_i\}_{i=1}^{N_y}$$
  • Definition (sample-based equivalence). We say that a sample set $\{\mathbf{x}_i\}_{i=1}^{N_x}$ is sample-based equivalent to $\{\mathbf{y}_i\}_{i=1}^{N_y}$ with respect to a set of feature functions $\{F_j(\mathbf{x})\}_{j=1}^{M}:\mathbb{R}^d\rightarrow\mathbb{R}$ if $$ \frac{1}{N_x}\sum_{i=1}^{N_x}F_j(\mathbf{x}_i) = \frac{1}{N_y}\sum_{j=1}^{N_y}F_j(\mathbf{y}_i),\quad j=1,2,\cdots, {M}$$ We denote this equivalence relation by $$\{\mathbf{x}_i\}_{i=1}^{N_x}\sim\{\mathbf{y}_i\}_{i=1}^{N_y}$$
  • The optimal map $\nabla\phi$ is constrained by
$$ \{\nabla\phi(\mathbf{x}_i)\}_{i=1}^{N_x}\sim\{\mathbf{y}_i\}_{i=1}^{N_y}$$

Sample-based Optimal Transport: $\nabla\phi$¶

  • Approximate the integral over measure by sample means:
$$\min_\phi\frac{1}{N_x}\sum_{i=1}^{N_x} \|\nabla\phi(\mathbf{x}_i) - \mathbf{x}_i\|^2$$
  • Extra constraints needs to be added on $\phi$.
  • We put $\phi$ in the space of all feature functions: $$ \phi(\mathbf{x}) = \frac{\|\mathbf{x}\|^2}{2} + \sum_{j=1}^{M} s_jF_j(\mathbf{x})$$
  • The $\frac{\|\mathbf{x}\|^2}{2}$ term is included so that when all the $s_j$ are zeros $\phi(\mathbf{x})$ corresponds to the identity map.

Sample-based Optimal Transport:
Comparison with Classical Optimal Transport

  • Correspondence:
$$ \begin{align} (\nabla\phi)_\sharp \mu_1 = \mu_2&\quad\rightarrow&\{\nabla\phi(\mathbf{x}_i)\}_{i=1}^{N_x}\sim\{\mathbf{y}_i\}_{i=1}^{N_y}\\ \int \| \nabla\phi(\mathbf{x})-\mathbf{x}\|^2 d\mu_1(\mathbf{x})&\quad\rightarrow& \frac{1}{N_x}\sum_{i=1}^{N_x} \|\nabla\phi(\mathbf{x}_i) - \mathbf{x}_i\|^2 \end{align} $$
  • Another correspondence: Monge-Ampere PDE,
$$ \begin{align} \int_{\mathbb{R}^d} h(\nabla\phi(\mathbf{x})) d\mu_1(\mathbf{x}) = \int_{\mathbb{R}^d} h(\mathbf{y})d\mu_2(\mathbf{y})&\quad\rightarrow& \frac{1}{N_x}\sum_{i=1}^{N_x}F_j(\nabla\phi(\mathbf{x}_i)) = \frac{1}{N_y}\sum_{j=1}^{N_y}F_j(\mathbf{y}_i)\end{align}$$
  • Feature functions are test functions!

Sample-based Optimal Transport¶

  • With input data $\{\mathbf{x}_i\}_{i=1}^{N_x}$ and $\{\mathbf{y}_i\}_{i=1}^{N_y}$, and a set of $C^1$ feature functions $\{F_j:\mathbb{R}^d\rightarrow\mathbb{R}\}_{j=1}^M$. The solution of the following constrained optimization problem of $s_1, s_2,\cdots, s_M$ defines the optimal map. $$ \begin{equation} \min_{\{\nabla\phi(\mathbf{x}_i)\}_{i=1}^{N_x}\sim\{\mathbf{y}_i\}_{i=1}^{N_y}}\sum_{i=1}^{N_x} \|\nabla\phi(\mathbf{x}_i) - \mathbf{x}_i\|^2, \end{equation} $$ where $$\phi(\mathbf{x}) = \frac{\|\mathbf{x}\|^2}{2} + \sum_{j=1}^{M} s_jF_j(\mathbf{x})$$

Local Optimal Transport Problem: Observation¶

  • $$\frac{1}{N_x}\sum_{i=1}^{N_x}F_j\left(\mathbf{x}_i + \sum_{j=1}^M s_j\nabla F_j(\mathbf{x}_i)\right) = \frac{1}{N_y}\sum_{i=1}^{N_y}F_j(\mathbf{y}_i),\quad j=1,\cdots,M$$
  • From the sample-base equivalence relation, we have $M$ unknowns and $M$ equations;
  • For general feature functions $F$, the system is nonlinear in $s_1,\cdots,s_M$ .
  • no solution? multiple solution? global optimization?

Local Optimal Transport Problem¶

  • When the two datasets are close, everything is good!
  • Denote
$$\mathbf{s} = (s_1,s_2,\cdots,s_M),\quad\mathbf{F}(\mathbf{x}) = (F_1(\mathbf{x}),F_2(\mathbf{x}),\cdots,F_M(\mathbf{x}))^T,$$$$ \mathbf{a} = \frac{1}{N_x}\sum_{i=1}^{N_x}\mathbf{F}(\mathbf{x}_i),\quad \mathbf{b} = \frac{1}{N_y}\sum_{i=1}^{N_y}\mathbf{F}(\mathbf{y}_i).$$
  • Theorem (local optimal transport). There exsits $\varepsilon$, when $\|\mathbf{a} - \mathbf{b}\|_{\infty} < \varepsilon$, we have
    1. The sample-based equivalence relation has a unique solution $\mathbf{s} = \mathbf{s}^*$ near $\mathbf{0}$;
    2. The corresponding $\phi^*$ function defined by $\mathbf{s}^*$ achives the global minimum of the sample-based optimal transport problem;
    3. If in addition all the feature functions are $C^2$ and their Hessian matrices are uniformly bounded, $\phi^*$ is convex.

Local Optimal Transport Problem: Remarks¶

  • We can define $$\mathbf{F}(\mathbf{x}) = (F_1(\mathbf{x}),F_2(\mathbf{x}),\cdots,F_M(\mathbf{x}))^T,$$ $$ \mathbf{G}(\mathbf{s}) = \frac{1}{N_x}\sum_{i=1}^{N_x}\mathbf{F}(\mathbf{x}_i + \mathbf{s}^T\nabla\mathbf{F}(\mathbf{x}_i))$$
  • The sample-based equivalence writes: $$\mathbf{G}(\mathbf{s}) = \mathbf{b}$$
  • We know $\mathbf{G}(\mathbf{0}) = \mathbf{a}$. Thus when $\mathbf{a}$ and $\mathbf{b}$ are close, inverse function theorem can help proving the existence.

Local Optimal Transport Problem: Remarks¶

  • Local sample-based optimal transport problem has great properties,
  • It is an existence theorem, but the size of $\varepsilon$ can be estimated,
  • One can solve a local sample-based optimal transport problem by solving the sample-based equivalence relation, a $M$ equation system, using standard nonlinear system solver.
  • We can naturally combine the theoretical algorithms with the local optimal transport solver. We will be able to derive solutions for general sample-based optimal transport and barycenter problem!

Optimal Transport and Bayrcenter Algorithms:
Sample-based Optimal Transport Algorithm (SOT)

  1. Let $\{\mathbf{x}_i^0\}_{i=1}^{N_x} = \{\mathbf{x}_i\}_{i=1}^{N_x}$ and $\{\mathbf{x}_i^K\}_{i=1}^{N_y} = \{\mathbf{y}_i\}_{i=1}^{N_y}$. Set initial sample sets $\{\mathbf{x}_i^k\}_{i=1}^{N_x}$ ($k=1,2,\cdots, K-1$);
  2. Select a set of feature functions $\{F_j(\mathbf{x})\}_{j=1}^M$;
  3. Set the current set $\{\mathbf{z}_i\}_{i=1}^{N_x}=\{\mathbf{x}_i^0\}_{i=1}^{N_x}$;
  4. For $k=1,2,\cdots, K$, solve the local sample-based optimal map $\mathbf{f}$ from $\{\mathbf{z}_i\}_{i=1}^{N_x}$ to $\{\mathbf{x}_i^k\}_{i=1}^{N_x}$ using a standard nonlinear system solver and update $\{\mathbf{z}_i\}_{i=1}^{N_x}$ through $$\begin{equation} \{\mathbf{z}_i\}_{i=1}^{N_x} = \{\mathbf{f}(\mathbf{z}_i)\}_{i=1}^{N_x}; \end{equation}$$
  5. For $k=1,2,\cdots, K-1$, update $\{\mathbf{x}_i^k\}_{i=1}^{N_x}$ through $$\begin{equation} \{\mathbf{x}_i^k\}_{i=1}^{N_x} = \{\frac{K - k}{K}\mathbf{x}_i^0 + \frac{k}{K}\mathbf{z}_i\}_{i=1}^{N_x} \end{equation}$$and go to step 3.

Optimal Transport and Bayrcenter Algorithms:
Composite Sample-based Barycenter Algorithm (CSB)

  1. Set the initial sample set $\{\mathbf{y}_i\}_{i=1}^{N_y}$;
  2. Select a set of feature functions $\{F_j(\mathbf{x})\}_{j=1}^M$;
  3. Run step 1,3,4 of SOT once for each pair $(\{\mathbf{y}_i\}_{i=1}^{N_y},\{\mathbf{x}_i^k\}_{i=1}^{N_k})$ to map $\{\mathbf{y}_i\}_{i=1}^{N_y}$ to a new set $\{\mathbf{z}_i^k\}_{i=1}^{N_y}$;
  4. Update $\{\mathbf{y}_i\}_{i=1}^{N_y}$ through $$\begin{equation} \{\mathbf{y}_i\}_{i=1}^{N_y} = \left\{\sum_{k=1}^K w_k\mathbf{z}_i^k\right\}_{i=1}^{N_y}. \end{equation}$$
  5. Run step 5, 3, 4 of SOT once for each pair $(\{\mathbf{y}_i\}_{i=1}^{N_y},\{\mathbf{x}_i^k\}_{i=1}^{N_k})$ to update $\{\mathbf{z}_i^k\}_{i=1}^{N_y}$ and go to step 4.

Optimal Transport and Bayrcenter Algorithms: Algorithm Parameters¶

  • Stop parameter $\varepsilon$ and stop criterion: $$\frac{1}{N_x}\sum_{i=1}^{N_x}\|\mathbf{z}_i^{\text{before}} - \mathbf{z}_i^{\text{after}}\|^2 <\varepsilon$$
  • Number of segments $M$, should be big enough so that each subproblem is local. In practice $M = 10\sim 50$;
  • Feature functions;

Selection of Feature Functions¶

  • The optimal map lives in the space of gradients of feature functions. The more feature functions one uses, the richer the structure of the optimal map;
  • Computational cost highly depends on feature functions: $M$ and computation of feature values.
  • Feature functions can be defined by the user, based on expert knowledge of the nature of the sample sets.

Selection of Feature Functions: Moments¶

  • Choosing for feature functions the components of $x$:
$$\begin{equation} F_j(\mathbf{x}) = \mathbf{x}^{(j)},\quad j = 1,2,\cdots, d. \end{equation}$$
  • The solution of sample-based optimal transport problem is a simple translation and matches the sample mean of two datasets.
  • Choosing for feature functions the first and second moments,
$$ \begin{align} F_j(\mathbf{x}) &= \mathbf{x}^{(j)},\quad j = 1,2,\cdots, d,\\ F_{\frac{(2n-i+1)i}{2} +j}(\mathbf{x})& = \mathbf{x}^{(i)}\mathbf{x}^{(j)},\quad 1\leq i\leq j\leq d. \end{align} $$

  • The sample-based optimal map is a linear map that matches the means and covariance matrices:
$$\nabla\phi(\mathbf{x}) = \mathbf{Ax}+\mathbf{b}.$$
  • It gives the true solutions to optimal transport and barycenter problems of normal distributions.
  • The corresponding basic barycenter algorithm gives a fixed-point iteration algorithm for barycenter problem of normal distributions (Agueh and Carlier 2011 and Alvarez-Esteban et al. 2016).

Selection of Feature Functions: Kernel Functions¶

  • For a given location $\mathbf{z}_0$ and a bandwidth parameter $h$, we can define a kernel function $F(\mathbf{x})$ such that $$ \begin{equation} F(\mathbf{x}) = K\left(\frac{\mathbf{x}-\mathbf{z}_0}{h}\right) \end{equation}$$
  • Corresponding sample feature mean is: $$\begin{equation} \frac{1}{N_x}\sum_{i=1}^{N_x}K\left(\frac{\mathbf{x}_i-\mathbf{z}_0}{h}\right), \end{equation}$$ which can be interpreted as the value of a kernel density estimator at point $\mathbf{z}_0$.
  • One natural candidate for the kernel function $K(\mathbf{x})$ is the Gaussian kernel $$ \begin{equation} K_h(\mathbf{x}) = \frac{1}{\sqrt{(2\pi h)^d}}\exp{\left(-\frac{\|\mathbf{x}\|^2}{2h^2}\right)}, \end{equation}$$

Selection of Feature Functions: Kernel Functions¶

  • Selection of a set of $\{(\mathbf{z}_j, h_j)\}_{j=1}^M$
  • $\mathbf{z}_j$ as grid points in the sample space;
  • Data-driven approaches: i.e. mean shift algorithm (Cheng 1995).

Selection of Feature Functions: Other Techniques¶

  • PCA, Kernel PCA
  • Nerual Networks
  • Prior Knowledge

Sample-based Perspective: Conclusions¶

  • Two algorithms are proposed:
    Sample-based Optimal Transport Algorithm (SOT)
    Composite Sample-based Barycenter Algorithm (CSB)
    Algorithms are set to be applied to optimal transport and barycenter problems with sample sets as inputs
  • Sample-based algorithms are extended from theoretical algorithm by combining with local optimal transport solvers.
  • Sample sets are compared through feature functions. Different sample-based solutions can be derived from different feature functions

Numerical Results: Normal Distributions¶

  • Optimal Transport of normal distributions has closed-form solution.
  • For multidimensional normal distributions $\mathcal{N}(\mathbf{m}_1,\mathbf{\Sigma}_1)$ and $\mathcal{N}(\mathbf{m}_2,\mathbf{\Sigma}_2)$, the optimal map will be a linear map. We have $$ \begin{equation} \mathbf{y} = \mathbf{m}_2 + (\mathbf{x}-\mathbf{m}_1) \mathbf{\Sigma}_1^{-1/2}(\mathbf{\Sigma}_1^{1/2}\mathbf{\Sigma}_2\mathbf{\Sigma}_1^{1/2})^{1/2}\mathbf{\Sigma}_1^{-1/2}. \end{equation} $$
  • It can be solved from the sample-based equivalence relation with first and second moments feature functions.

Applications: Shape Transform¶

  • The task is to find maps or barycenters for different shapes.
  • Optimal Transport provides a good metric to between arbitrary shapes. Choosing $L_2$ cost, the sample-based algortihms gives:
    1. Optimal map
    2. All the intermediate shapes
    3. One to one correspondence of arbitrary points
  • Challenges: Sharp boundary, Different topology, Shape representations

Shape Transform: Technical Details¶

  • For a bounded region (shape) $\Omega\in\mathbb{R}^d$, we define a grid in $\mathbb{R}^d$ and choose grid points $\{\mathbf{x}_i\}$ in $\Omega$ to represent the shape.
  • From an arbitray sample set $\{\mathbf{x}_i\}$, we first derive a density estimator $\tilde{p}$ and define the shape to be regions with $\tilde{p}(\mathbf{x}) > \varepsilon$.
  • Algorithm parameters
    1. Feature functions are choosen to be gaussian kernels on a $7\times 7$ grid with bandwidth $h = 0.5, 1$. ($49$ feature functions).
    2. $K = 20$, $\epsilon = 10^{-4}$.

Color Transfer: The Meta-algorithm¶

  • To do good color transfer, optimal transport is not enough. We adopt Rabin et al (2015)'s meta-algorithm:
  • Spatio-color Clustering: cluster the image into subsets, which builds a smaller sample set
  • Sample-based optimal transport algorithm
  • Image synthethesis: recover color of each pixel from cluster sample sets
  • Post-processing: restore sharp details of the original image

  • Spatio-color clustering: super-pixel method, 2000 clusters, CIELAB space
  • Sample-based optimal transport: SOT, $K = 20, 5\times 5\times 5$ grid Gaussian kernels, $h = 1$
  • Image synthesis: Special expectation-maximization algorithm
  • Post-processing: iterative TMR filters (Rabin et al. 2011)
